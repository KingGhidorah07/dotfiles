# 🐉 KingGhidorah Dotfiles

Setup de Ubuntu 26.04 con tema Catppuccin Mocha.

## Incluye
- Zsh + Oh My Zsh + Starship
- Kitty terminal
- Fastfetch con Ghidorah ASCII
- btop
- Wallpaper switcher automático

## Instalación
```bash
cp .zshrc ~/
cp starship.toml ~/.config/
cp -r kitty ~/.config/
cp -r fastfetch ~/.config/
cp -r btop ~/.config/
cp wallpaper-switcher.sh ~/.local/bin/
chmod +x ~/.local/bin/wallpaper-switcher.sh
```
