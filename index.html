Ubuntu Setup Guide

De Ubuntu stock a un setup pro con tema Catppuccin Mocha

Ubuntu 24.04+ · GNOME · Catppuccin Mocha
1
Terminal Pro Zsh + Oh My Zsh + Starship + Kitty
01
Instalar Zsh y cambiarlo como shell por defecto
sudo apt update && sudo apt install zsh -y chsh -s $(which zsh)
Nota: Cierra sesión y vuelve a entrar para que tome efecto.
02
Instalar Oh My Zsh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
03
Instalar plugins de Zsh
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

Luego edita ~/.zshrc y cambia la línea de plugins:

plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
04
Instalar Starship prompt
curl -sS https://starship.rs/install.sh | sh starship preset catppuccin-powerline -o ~/.config/starship.toml

Agrega al final de ~/.zshrc:

eval "$(starship init zsh)"
05
Instalar Kitty + Nerd Font
sudo apt install kitty -y mkdir -p ~/.local/share/fonts && cd ~/.local/share/fonts curl -fLo "JetBrainsMono.zip" https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/JetBrainsMono.zip unzip JetBrainsMono.zip && rm JetBrainsMono.zip && fc-cache -fv
2
Tema del Sistema Catppuccin GTK + Iconos Papirus
01
Instalar herramientas
sudo apt install gnome-tweaks gnome-shell-extensions papirus-icon-theme -y
02
Descargar y aplicar tema Catppuccin GTK
mkdir -p ~/.themes && cd ~/.themes curl -fLo catppuccin.zip https://github.com/catppuccin/gtk/releases/download/v1.0.3/catppuccin-mocha-mauve-standard+default.zip unzip catppuccin.zip && rm catppuccin.zip gsettings set org.gnome.desktop.interface gtk-theme "catppuccin-mocha-mauve-standard+default" gsettings set org.gnome.desktop.interface icon-theme "Papirus-Dark"
3
Extensiones GNOME Blur, Rounded corners, Animaciones
01
Instalar Extension Manager
sudo apt install gnome-shell-extension-manager -y
02
Instalar estas extensiones desde Extension Manager
Blur my Shell Rounded Window Corners Reborn User Themes Burn My Windows Compiz alike magic lamp Transparent Top Bar Vitals Overview Background Dash to Dock
03
Aplicar tema al Shell
gsettings set org.gnome.shell.extensions.user-theme name "catppuccin-mocha-mauve-standard+default"
4
VS Code Tema + Fuente + Extensiones
01
Extensiones a instalar en VS Code
Catppuccin for VS Code Catppuccin Icons Prettier GitLens Error Lens Auto Rename Tag
02
Configuración en Settings (Ctrl+,)
  • Font Family → 'JetBrainsMono Nerd Font Mono', monospace
  • Font Size → 14
  • Font Ligatures → activado
  • Color Theme → Catppuccin Mocha
  • File Icon Theme → Catppuccin Mocha
  • Terminal Default Profile → zsh
5
Wallpaper Automático Cambia según la hora del día
01
Descargar wallpapers
mkdir -p ~/Imágenes/wallpapers && cd ~/Imágenes/wallpapers wget -O wall1.jpg "https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=1920" wget -O wall2.jpg "https://images.unsplash.com/photo-1464822759023-fed622ff2c3b?w=1920" wget -O wall3.jpg "https://images.unsplash.com/photo-1500534314209-a25ddb2bd429?w=1920" wget -O wall4.jpg "https://images.unsplash.com/photo-1426604966848-d7adac402bff?w=1920" wget -O wall5.jpg "https://images.unsplash.com/photo-1472214103451-9374bd1c798e?w=1920" wget -O wall6.jpg "https://images.unsplash.com/photo-1518173946687-a4c8892bbd9f?w=1920"
02
Crear script y cron
mkdir -p ~/.local/bin cat > ~/.local/bin/wallpaper-switcher.sh << 'EOF' #!/bin/bash WALL_DIR="$HOME/Imágenes/wallpapers" HOUR=$(date +%H) if [ $HOUR -ge 5 ] && [ $HOUR -lt 8 ]; then WALL="$WALL_DIR/wall1.jpg" elif [ $HOUR -ge 8 ] && [ $HOUR -lt 12 ]; then WALL="$WALL_DIR/wall2.jpg" elif [ $HOUR -ge 12 ] && [ $HOUR -lt 15 ]; then WALL="$WALL_DIR/wall3.jpg" elif [ $HOUR -ge 15 ] && [ $HOUR -lt 18 ]; then WALL="$WALL_DIR/wall4.jpg" elif [ $HOUR -ge 18 ] && [ $HOUR -lt 20 ]; then WALL="$WALL_DIR/wall5.jpg" else WALL="$WALL_DIR/wall6.jpg"; fi gsettings set org.gnome.desktop.background picture-uri "file://$WALL" gsettings set org.gnome.desktop.background picture-uri-dark "file://$WALL" EOF chmod +x ~/.local/bin/wallpaper-switcher.sh (crontab -l 2>/dev/null; echo "0 * * * * DBUS_SESSION_BUS_ADDRESS=unix:path=/run/user/$(id -u)/bus ~/.local/bin/wallpaper-switcher.sh") | crontab -
6
Fastfetch + btop + Aliases El flex final
01
Instalar fastfetch y btop
sudo apt install fastfetch btop -y echo "fastfetch" >> ~/.zshrc
02
Aliases útiles — agregar a ~/.zshrc
alias ..="cd .." alias ll="ls -lah" alias gs="git status" alias ga="git add ." alias gc="git commit -m" alias gp="git push" alias gl="git log --oneline --graph" alias update="sudo apt update && sudo apt upgrade -y" alias myip="curl ifconfig.me" alias serve="python3 -m http.server 8000"
03
Atajos de teclado
